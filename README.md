# DUMP Days 2021 Website Recreation

### Repository for DUMP Internship: Week 10 - CSS #2

## Task

This project is a **responsive recreation** of the DUMP Days 2021 website:  
https://2021.days.dump.hr/

The goal is to replicate the **visual design, layout, and animations** from the provided **Figma file**.

The design includes both **desktop (1440px and above)** and **mobile (375px)** versions, and the website should be **fully responsive** across all screen sizes.

## Setup

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
